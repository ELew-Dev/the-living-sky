* **DEFINE:** I need to create a function called `project_motion(df, years)`.
* **INPUT:** Insert the Big Dipper data and the number of years.

### **Technical Deep Dive: Why Pandas?**
* **The Dataframe (`df`):** This is the core object of the Pandas library. The rows contain individual stars, while columns store names, coordinates, and velocities.
* **ESSENTIAL:** Without a `df`, if you wanted to change 100,000 stars, there would need to be a loop written that would change it one by one. 
* **Vectorization:** If I write `df['ra'] + 10`, Pandas will update all 100k rows at the exact same time. This is the difference between a script that hangs and one that actually works.
